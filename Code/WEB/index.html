Ce projet a pour objectif de recréer informatiquement le système solaire.
Les calculs des positions et des vitesse des planètes sont calculé via un programme écrit en C, qui va par la suite exporter toutes ces données dans un fichier au format JSON qui est un format lisible et traitable par le Javascript. Ensuite le but est de récupérer ces données via un script Javascript pour les traiter afin d'afficher les trajectoires ainsi que les planètes se déplaçant en orbite autour du soleil. La maquette sera présentée sur une page web qui sera dressée par du HTML/CSS.
